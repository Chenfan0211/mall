# 商城项目功能清单

编制日期：2026-06-09

## 1. 功能模块总览

| 序号 | 模块 | 所属端 | 说明 |
| --- | --- | --- | --- |
| 1 | 商品中心 | 运营后台 | 商品基础资料、规格、库存规格、价格、联想词、ES 同步 |

| 7 | 售后中心 | 运营后台/用户小程序/自提点/供应商 | 仅退款、退货退款、部分退款、售后状态流转 |

| 15 | WMS 出库 | 仓配小程序/H5/后台 | 销售单、波次下放、拣货、复检、装车、配送 |
| 16 | 库存管理 | 仓配小程序/H5/后台 | 实时库存、库存流水、FIFO、库位、移库、报损、盘点 |
| 17 | 系统任务 | 服务端 | 定时任务、消息队列、缓存刷新、ES 同步、公众号通知 |

## 2. 运营后台功能

### 2.1 商品中心

| 功能 | 说明 |
| --- | --- |
| 商品基础信息 | 维护商品名称、图片、描述、供应商、采购员、类目等基础信息 |
| 商品规格 | 一个商品支持多个规格 |
| 库存规格绑定 | 商品规格可绑定库存规格 |
| 规格系数转换 | 支持商品规格与库存规格之间的换算关系 |
| 城市价格 | 每个商品维护三级城市价格信息 |
| 商品联想词 | 创建商品时自动生成联想词，支持自定义新增和修改 |
| ES 同步 | 商品新增、修改后同步到 Elasticsearch |
| 商品售罄想要 | 记录用户“想要”，商品重新上架后可通知用户 |

### 2.2 类目中心

| 功能 | 说明 |
| --- | --- |
| 展示类目管理 | 创建、编辑、删除展示类目，支持多级 |
| 统计类目管理 | 创建、编辑、删除统计类目，支持多级 |
| 商品绑定展示类目 | 商品用于前台展示和筛选 |
| 商品绑定统计类目 | 商品用于报表统计和经营分析 |

### 2.3 基础资料

| 功能 | 说明 |
| --- | --- |
| 供应商管理 | 创建供应商资料，并生成供应商登录账号 |
| 自提点管理 | 创建自提点，绑定城市，并生成自提点登录账号 |
| 采购员管理 | 创建采购员资料，并生成采购员登录账号 |
| 城市管理 | 维护一二三级城市 |
| 城市截团时间 | 每个城市配置截团时间 |
| 仓库管理 | 创建采购仓/配送仓，并生成仓库登录账号 |

### 2.4 商品发布

| 功能 | 说明 |
| --- | --- |
| 发布模板 | 通过模板发布商品 |
| 配送日期指定 | 支持手动指定配送日期 |
| 配送日期校验 | 配送日期必须大于销售结束时间 |
| 自动计算配送日 | 未指定配送日时系统自动计算 |
| 城市白名单 | 指定某些城市可售 |
| 自提点白名单 | 指定某些自提点可售 |
| 城市价格调整 | 发布时可手动调整商品城市价格 |
| 指定采购仓 | 发布团购必须指定采购仓，衔接采购流程 |

### 2.5 权限中心

| 功能 | 说明 |
| --- | --- |
| 用户登录 | 支持平台、城市、供应商、采购员、自提点、仓库等角色登录 |
| 菜单权限 | 按角色控制后台菜单和操作权限 |
| 数据权限 | 按城市、供应商、采购员、仓库、自提点进行数据隔离 |
| 角色管理 | 支持预设角色和自定义角色 |
| 权限配置 | 支持功能权限、数据权限、端口权限配置 |

### 2.6 订单中心

| 功能 | 说明 |
| --- | --- |
| 订单列表 | 查看用户订单，按状态、城市、自提点、商品筛选 |
| 订单详情 | 查看商品、价格、优惠、支付、配送、售后信息 |
| 订单状态流转 | 未支付、已付款、取消、已发货、已收货、已完成 |
| 支付流水 | 支付成功后生成支付流水 |
| 未支付取消 | 15 分钟未支付自动取消，支持用户手动取消 |
| 发货后限制 | 订单发货后不允许取消，只能走售后 |

### 2.7 售后中心

| 功能 | 说明 |
| --- | --- |
| 售后申请列表 | 查看用户提交的售后单 |
| 售后审核 | 审核仅退款或退货退款 |
| 部分退款 | 支持订单部分退款 |
| 仅退款 | 不需要退货时审核后直接退款 |
| 退货退款 | 退货完成后才能退款 |
| 售后状态 | 待审核、待退货、待退款、退款成功、退款失败等 |

### 2.8 分账中心

| 功能 | 说明 |
| --- | --- |
| 分账方账户 | 管理自提点、供应商等分账方佣金账户 |
| 佣金明细 | 查看订单佣金明细 |
| 提成流水 | 记录佣金产生过程 |
| 分账流水 | 订单完成后生成分账流水 |
| 提现申请 | 分账方提交提现申请 |
| 财务审核 | 后台审核提现申请并处理打款 |

### 2.9 营销中心

| 功能 | 说明 |
| --- | --- |
| 优惠券管理 | 创建、发放、停用优惠券 |
| 发放黑白名单 | 指定城市或自提点可发放/不可发放 |
| 使用黑白名单 | 指定商品可使用/不可使用优惠券 |
| 活动商品页 | 指定某些商品放在同一活动页面 |
| 推荐规则 | 根据用户喜好推荐商品 |
| 首页弹券 | 用户进入小程序时自动弹出优惠券领取 |

### 2.10 报表中心

| 功能 | 说明 |
| --- | --- |
| 订单报表 | 按时间、城市、自提点、商品统计订单 |
| 分佣报表 | 各分账方订单分佣统计 |
| 配送商品汇总报表 | 按配送日、城市、自提点、商品汇总配送数量 |
| 统计类目销售报表 | 按统计类目汇总销售数据 |
| 售后订单报表 | 统计售后数量、金额、退款状态 |
| 库存报表 | 查询实时库存、库存流水、收发存数据 |

## 6. WMS 功能清单

### 6.1 采购管理

| 功能 | 说明 |
| --- | --- |
| 采购订单创建 | 选择供应商、仓库、商品规格，录入采购数量和单价 |
| 采购订单审核 | 审核后锁定采购订单，并生成关联入库单 |
| 收货状态跟踪 | 查看入库单、已收数量、待收数量、收货状态 |
| 已审核印章 | 采购订单详情展示已审核标识 |
| 打印采购订单 | 支持采购订单打印或 PDF 导出 |
| 供应商退货单 | 创建供应商退货单，审核后按 FIFO 扣减库存 |

### 6.2 收货管理

| 功能 | 说明 |
| --- | --- |
| 入库单列表 | 按仓库、状态、供应商过滤入库单 |

| 收货管理只读 | 后台查看入库进度和收货记录 |

### 6.3 销售单与波次下放

## 7. 自动化与中间件功能

| 功能 | 技术建议 | 说明 |
| --- | --- | --- |
| 商品 ES 同步 | RocketMQ + Elasticsearch | 商品新增/修改后异步同步 |
| 商品列表缓存 | Redis | 缓存首页、类目、活动商品列表 |
| 定时刷新 | XXL-Job | 定时刷新商品列表、检测库存、取消订单 |
| 未支付超时 | RocketMQ 延迟消息或 XXL-Job | 15 分钟未支付自动取消 |
| 支付提醒 | XXL-Job + 公众号模板消息 | 最后 5 分钟提醒用户支付 |
| 佣金通知 | RocketMQ + 公众号模板消息 | 支付成功后通知自提点 |
| 服务注册配置 | Nacos | Spring Cloud 服务注册、配置管理 |
| 分布式缓存 | Redis | 登录态、验证码、热点数据、锁 |
| 容器部署 | Docker | 后端服务、中间件、前端静态资源容器化 |

## 8. 数据权限隔离建议

| 角色 | 数据范围 |
| --- | --- |
| 平台管理员 | 全部数据 |
| 城市角色 | 当前城市及下级城市数据 |
| 供应商 | 当前供应商商品、订单、售后、佣金数据 |
| 采购员 | 自己负责商品、采购、入库相关数据 |
| 自提点 | 当前自提点订单、配送、佣金、售后数据 |
| 仓库角色 | 当前仓库及授权库区数据 |
| 司机 | 当前司机配送任务数据 |
