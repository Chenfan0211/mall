# 功能清单-商品中心

## 模块名称

商品中心首批编码模块，对应 `mall-product`。

## 功能列表

| 功能名称 | 用途描述 | 请求路径 | 入参说明 | 出参说明 |
|----------|----------|----------|----------|----------|
| 查询商品列表 | 运营后台商品中心按关键词、供应商、类目、审核状态、启停状态、城市筛选商品 | `GET /api/product/products` | `keyword`、`supplierId`、`frontCategoryId`、`statCategoryId`、`auditStatus`、`saleStatus`、`cityId`、`pageNum`、`pageSize` | 商品分页列表 |
| 查看商品详情 | 查看商品基础资料、类目、供应商、主图、审核状态和启停状态 | `GET /api/product/products/{id}` | 商品ID | 商品详情 |
| 查询类目列表 | 查询前台展示类目和后台统计类目基础数据 | `GET /api/product/categories` | `categoryType`、`parentId`、`status` | 类目列表 |
| 查询 SKU 列表 | 按商品、供应商、状态查询 SKU 基础资料 | `GET /api/product/skus` | `productId`、`supplierId`、`status` | SKU 列表 |
| 查看 SKU 详情 | 查看 SKU 编码、销售规格、条码、单位、供应商和状态 | `GET /api/product/skus/{id}` | SKU ID | SKU 详情 |
| 查询 SKU 库存规格 | 查看 SKU 绑定的库存规格、换算系数和默认标识 | `GET /api/product/skus/{id}/stock-specs` | SKU ID | 库存规格列表 |

## 业务规则

| 规则 | 说明 |
|------|------|
| 商品资料边界 | 商品中心只维护基础资料、SKU、类目和规格口径；实际售卖价格、库存模式、销售时间由团期决定 |
| 类目口径 | 前台展示类目用于小程序分类，统计类目用于售后规则、报表统计和经营分析 |
| SKU 口径 | SKU 是库存、采购、订单商品、售后、分佣和报表统计的最小商品单位 |
| 规格绑定 | SKU 必须能追溯库存规格和换算系数；第一批接口先提供只读查询 |
| 数据隔离 | `prd_product` 按城市、供应商范围隔离；`prd_sku` 按供应商范围隔离；类目等基础数据默认可见 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用 `prd_category`、`prd_product`、`prd_sku`、`prd_stock_spec`、`prd_sku_stock_spec` 表和初始化数据 |
| `mall-common` | Result、分页、基础实体、全局异常、网关认证头、数据权限拦截器 |
| `mall-api` | 商品、类目、SKU VO/DTO 契约 |
| MyBatis Plus | 单表查询、分页和数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
