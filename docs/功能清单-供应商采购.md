# 功能清单-供应商采购

## 模块名称

供应商采购首批编码模块，对应 `mall-supplier`。

## 业务规则

| 规则 | 说明 |
|------|------|
| 只读起步 | 本批只提供查询能力，不实现供应商审核、资质审核、合同维护、采购审核、送货确认和到仓收货 |
| 状态完整 | 初始化数据覆盖供应商正常/停用/拉黑、资质待审/通过/驳回、合同待生效/生效/到期/解除、采购和送货全关键状态 |
| 数据隔离 | 供应商主体按城市和自身供应商隔离；采购单按城市、供应商、仓库隔离；送货单按供应商、仓库以及关联采购城市隔离 |
| 基础数据可见 | 城市、仓库、类目等基础数据仍由基础模块提供，供应商模块只返回业务事实表数据 |
| 首页聚合 | 供应商工作台首页只做读侧汇总，不改变采购、送货、收货或财务状态 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用供应商主体、联系人、资质、合同、采购单、采购明细、采购审核日志、送货单、送货明细、采购来源记录表和初始化数据 |
| `mall-common` | Result、分页、基础实体、全局异常、网关认证头、数据权限拦截器 |
| `mall-api` | 供应商、采购、送货 VO/DTO 契约 |
| MyBatis Plus | 单表查询、分页和数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
