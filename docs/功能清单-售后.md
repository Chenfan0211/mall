# 功能清单-售后

## 模块名称

售后首批编码模块，对应 `mall-aftersale`。

## 功能列表

| 功能名称 | 用途描述 | 请求路径 | 入参说明 | 出参说明 |
|----------|----------|----------|----------|----------|
| 查询售后单列表 | 按售后单号、订单号、用户、城市、自提点、售后类型、状态、责任方筛选售后单 | `GET /api/aftersale/after-sales` | `keyword`、`orderId`、`orderNo`、`userId`、`cityId`、`stationId`、`afterSaleType`、`status`、`responsibilityType`、`pageNum`、`pageSize` | 售后单分页列表 |
| 查看售后单详情 | 查看售后单号、订单、申请原因、退款金额、责任方和当前状态 | `GET /api/aftersale/after-sales/{id}` | 售后单ID | 售后单详情 |
| 查询售后审核日志 | 查看售后审核动作、前后状态、审核人和审核原因 | `GET /api/aftersale/after-sales/{id}/audit-logs` | 售后单ID | 售后审核日志列表 |
| 查询售后商品列表 | 按售后单、订单商品、SKU、供应商筛选售后商品明细 | `GET /api/aftersale/items` | `afterSaleId`、`orderItemId`、`skuId`、`supplierId`、`pageNum`、`pageSize` | 售后商品分页列表 |
| 查看售后商品详情 | 查看申请数量、审核通过数量、退款金额和供应商责任证据 | `GET /api/aftersale/items/{id}` | 售后商品ID | 售后商品详情 |
| 查询退货记录列表 | 按退货编号、售后单、订单商品、SKU、自提点、仓库、退货状态筛选退货记录 | `GET /api/aftersale/return-records` | `keyword`、`afterSaleId`、`orderItemId`、`skuId`、`stationId`、`warehouseId`、`returnStatus`、`pageNum`、`pageSize` | 退货记录分页列表 |
| 查看退货记录详情 | 查看退货数量、退货状态、回仓仓库和凭证 | `GET /api/aftersale/return-records/{id}` | 退货记录ID | 退货记录详情 |

## 业务规则

| 规则 | 说明 |
|------|------|
| 只读起步 | 本批只提供查询能力，不实现审核、驳回、退款、退款重试、补货、责任方认定、佣金扣回等高风险动作 |
| 商品维度 | 售后按订单商品维度追踪，订单商品履约状态不混入售后状态 |
| 退货退款 | 退货退款需要退货记录作为退款前置证据，本批只展示记录，不处理仓库回仓入库 |
| 责任追溯 | 售后单、售后商品、审核日志、退货记录作为责任方认定、退款、佣金扣回的追溯依据 |
| 数据隔离 | 售后主单按城市、自提点、供应商和仓库关联范围隔离；售后商品按供应商、城市、自提点、仓库范围隔离；审核日志和退货记录通过售后单或售后商品反查隔离 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用 `afs_after_sale`、`afs_after_sale_item`、`afs_audit_log`、`afs_return_record` 表和初始化数据 |
| `mall-common` | Result、分页、基础实体、全局异常、网关认证头、数据权限拦截器 |
| `mall-api` | 售后单、售后商品、审核日志、退货记录 VO/DTO 契约 |
| MyBatis Plus | 单表查询、分页和数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
