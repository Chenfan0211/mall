# 功能清单-网关

## 模块名称

统一网关首批编码模块，对应 `mall-gateway`。

## 功能列表

| 功能名称 | 用途描述 | 请求路径 | 入参说明 | 出参说明 |
|----------|----------|----------|----------|----------|
| 认证服务路由 | 转发账号密码登录等认证接口 | `/api/auth/**` | 原始 HTTP 请求 | 转发到 `mall-auth` |
| 系统服务路由 | 转发账号、角色、菜单、字典等系统接口 | `/api/system/**` | 原始 HTTP 请求 | 转发到 `mall-system` |
| 运营服务路由 | 转发待办与异常中心接口 | `/api/operation/**` | 原始 HTTP 请求 | 转发到 `mall-operation` |
| 商品服务路由 | 转发商品、SKU、类目等商品中心接口 | `/api/product/**` | 原始 HTTP 请求 | 转发到 `mall-product` |
| 团期服务路由 | 转发团期列表、团期 SKU、快照等团期管理接口 | `/api/sale/**` | 原始 HTTP 请求 | 转发到 `mall-sale` |
| 交易服务路由 | 转发订单中心、订单商品、履约跟踪和缺货记录接口 | `/api/trade/**` | 原始 HTTP 请求 | 转发到 `mall-trade` |
| 支付服务路由 | 转发支付单、退款单和资金流水接口 | `/api/payment/**` | 原始 HTTP 请求 | 转发到 `mall-payment` |
| 售后服务路由 | 转发售后单、售后商品、审核日志和退货记录接口 | `/api/aftersale/**` | 原始 HTTP 请求 | 转发到 `mall-aftersale` |
| 供应商采购路由 | 转发供应商档案、采购单和送货单接口 | `/api/supplier/**` | 原始 HTTP 请求 | 转发到 `mall-supplier` |
| 财务结算路由 | 转发财务账户、佣金分账、提现押金、结算和对账接口 | `/api/finance/**` | 原始 HTTP 请求 | 转发到 `mall-finance` |
| 仓储履约路由 | 转发仓库基础资料、入库、库存、出库配送和回仓接口 | `/api/wms/**` | 原始 HTTP 请求 | 转发到 `mall-wms` |
| 角色工作台路由 | 转发自提点/团长工作台、订单、到货、休假、资金、消息接口 | `/api/station/**` | 原始 HTTP 请求 | 转发到 `mall-station` |
| 用户小程序路由 | 转发首页、商品、购物车、订单、售后、自提点、消息接口 | `/api/user/**` | 原始 HTTP 请求 | 转发到 `mall-user` |
| 跨域配置 | 支持后台和 H5 本地联调跨域访问 | 全部路径 | Origin、Method、Header | Gateway CORS 响应头 |

## 业务规则

| 规则 | 说明 |
|------|------|
| 统一入口 | 本地默认端口 `18000`，前端优先访问网关，不直接调用下游服务 |
| 路由边界 | 当前接入 `mall-auth`、`mall-system`、`mall-operation`、`mall-product`、`mall-sale`、`mall-trade`、`mall-payment`、`mall-aftersale`、`mall-supplier`、`mall-finance`、`mall-wms`、`mall-station`、`mall-user`，其他业务模块实现后再追加 |
| 鉴权透传 | 下游服务已支持 `X-Mall-*` 认证和数据范围头；JWT/Redis 会话解析在后续网关鉴权切面补齐 |
| 暂缓能力 | 网关不暴露积分、优惠券、客服、复杂 BI 等第一版暂缓模块路由 |
