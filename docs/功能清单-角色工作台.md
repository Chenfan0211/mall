# 功能清单-角色工作台

## 模块名称

角色工作台自提点/团长首批编码模块，对应 `mall-station`。

## 业务规则

| 规则 | 说明 |
|------|------|
| 数据隔离 | 自提点账号只能看当前自提点数据；团长账号只能看绑定团长及站点数据；基础资料按授权城市/站点/团长范围可见 |
| 提货核销 | 只有待自提订单商品可以核销；支持部分核销，全部核销后商品状态变为已自提；同一订单全部商品已自提后订单履约状态同步完成 |
| 到货确认 | 只有已到达配送自提点可以确认完成；同一配送单下所有站点完成或无法送达后配送单同步完成 |
| 休假申请 | 同一自提点/团长在待审核或已通过时间段内不允许重复交叉申请；只有待审核申请可以撤回 |
| 提现申请 | 只有正常账户且可用余额充足才能申请；提交后冻结可用余额并写入资金流水；只有待审核提现可以撤回并解冻 |
| 到货口径 | 自提点到货以 WMS 配送站点和配送单为依据，司机签到默认形成到站节点 |
| 资金口径 | 佣金、账户、提现复用财务表；订单完成后才进入最终分账口径 |
| 消息口径 | 消息按接收方类型和接收方ID隔离，不展示其他自提点、团长或供应商消息 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用自提点、团长、订单、WMS 配送、财务、消息、异常等已建表 |
| `mall-common` | 复用统一返回、分页、异常、认证头和 MyBatis 数据权限拦截器 |
| `mall-gateway` | 通过 `/api/station/**` 路由到 `mall-station` |
