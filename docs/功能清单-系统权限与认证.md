# 功能清单-系统权限与认证

## 模块名称

系统权限与认证首批编码模块，包含 `mall-common`、`mall-api`、`mall-system`、`mall-auth`。

## 功能列表

| 功能名称 | 用途描述 | 请求路径 | 入参说明 | 出参说明 |
|----------|----------|----------|----------|----------|
| 账号密码登录 | 按账号、密码、登录端校验账号状态并返回账号、角色、菜单、按钮权限、数据范围 | `POST /api/auth/password-login` | `username`、`password`、`portalCode` | 登录Token、账号信息、角色列表、菜单列表、按钮权限、数据范围 |
| 查询后台账号 | 运营后台查看账号列表 | `GET /api/system/accounts` | `keyword`、`loginPortalCode`、`accountType`、`status`、`pageNum`、`pageSize` | 账号分页列表 |
| 查看账号详情 | 查看单个账号基础信息 | `GET /api/system/accounts/{id}` | 账号ID | 账号详情 |
| 查询角色列表 | 查看角色配置 | `GET /api/system/roles` | `portalCode`、`status`、`pageNum`、`pageSize` | 角色分页列表 |
| 查询账号角色 | 查看账号绑定角色 | `GET /api/system/roles/account/{accountId}` | 账号ID | 角色列表 |
| 查询菜单列表 | 查看指定端可见菜单 | `GET /api/system/menus` | `portalCode` | 菜单列表 |
| 查询字典项 | 查看状态字典和基础字典项 | `GET /api/system/dicts/items` | `dictCode`、`status` | 字典项列表 |

## 业务规则

| 规则 | 说明 |
|------|------|
| 账号统一入口 | 运营后台、仓库后台、仓配H5、角色工作台小程序统一使用 `sys_account` |
| 登录端隔离 | 登录时必须传 `portalCode`，账号只能在初始化配置的登录端登录 |
| 状态校验 | 只有 `status=1` 的账号允许继续密码校验，禁用或锁定账号直接拒绝 |
| 权限聚合 | 登录成功后聚合 `sys_account_role`、`sys_role_menu`、`sys_account_data_scope` |
| 按钮权限 | 登录成功后按 `sys_role_button` 拼接 `module_code:button_code`，用于网关透传和 `@PreAuthorize` 校验 |
| 数据隔离 | 登录成功返回数据范围；后台服务支持通过 `X-Mall-*` 网关头写入 `DataScopeContext` |
| 密码哈希 | `Sql/mall.sql` 末尾已追加测试账号 BCrypt 哈希回填 SQL，执行完整脚本后可直接登录 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用 `sys_account`、`sys_role`、`sys_menu`、`sys_dict_item`、`sys_data_scope` 等表 |
| MyBatis Plus | 单表 CRUD、分页查询 |
| MapStruct | Entity 到 DTO 转换 |
| Spring Security | 密码编码器、`@PreAuthorize` 权限注解 |
| MySQL 8 | 系统权限和认证数据存储 |
