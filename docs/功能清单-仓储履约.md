## 模块名称

## 业务规则

| 规则 | 说明 |
|------|------|
| 只读联调 | 首批仅开放查询和详情接口，收货、上架、锁库存、拣货、装车、出库、签到、回仓确认等写操作待业务校验闭环后再实现 |
| 数据隔离 | 数据范围统一走 `MallDataPermissionHandler`，业务 Service 不手写数据过滤 |
| 仓库范围 | 仓库角色按 `warehouse_id` 查看当前仓库数据 |
| 城市范围 | 城市运营通过仓库所属 `city_id` 查看城市仓储数据 |
| 司机范围 | 司机范围使用司机账号 ID，限制配送、签到、回仓等司机任务数据 |
| 自提点范围 | 自提点角色仅能查看当前自提点相关的配送、装车、签到和回仓数据 |
| 数量字段 | 数量、状态、ID 等整数统一使用 `Long` / SQL `BIGINT` |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `mall-common` | 统一返回、分页、异常、网关认证、数据权限拦截器 |
| `mall-api` | WMS DTO/VO 定义 |
| MyBatis Plus | 单表分页、逻辑删除、数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
| `mall-gateway` | `/api/wms/**` 路由到 `mall-wms` |
| MySQL | 复用 `Sql/mall.sql` 中已有 WMS 表和测试数据 |
