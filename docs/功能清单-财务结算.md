# 功能清单-财务结算

## 模块名称

财务结算首批编码模块，对应 `mall-finance`。

## 业务规则

| 规则 | 说明 |
|------|------|
| 只读起步 | 本批只提供查询能力，不实现佣金重算、分账发起、提现审核、提现打款、打款重试、押金释放和对账处理 |
| 数量类型 | 账户笔数、对账笔数、状态、主体类型等整数类字段统一使用 `BIGINT` / `Long` |
| 金额事实 | 账户余额、佣金、分账、提现、押金、结算和对账金额使用 `DECIMAL` / `BigDecimal`，不允许用浮点类型 |
| 状态覆盖 | 初始化数据覆盖账户正常/冻结、佣金待计算/待分账/已分账/扣回、分账待处理/处理中/成功/失败、提现待审/通过/驳回/撤回/成功/失败、押金冻结/部分释放/已释放、供应商结算待确认/已确认/已入账/异常、对账待处理/处理中/完成/异常 |
| 数据隔离 | 账户、流水、提现、押金按主体隔离；佣金和分账按主体及关联订单/订单商品隔离；供应商结算按供应商隔离；对账批次仅平台全量角色可见 |
| 基础数据可见 | 角色、城市、供应商、自提点、团长等基础资料由系统、用户和供应商模块提供，财务模块只返回资金事实表数据 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用 `fin_account`、`fin_account_flow`、`fin_commission_detail`、`fin_split_flow`、`fin_withdraw_apply`、`fin_deposit_record`、`fin_supplier_settlement`、`fin_reconciliation_batch` 表和初始化数据 |
| `mall-common` | Result、分页、基础实体、全局异常、网关认证头、数据权限拦截器 |
| `mall-api` | 财务账户、佣金、分账、提现、押金、结算、对账 VO/DTO 契约 |
| MyBatis Plus | 单表查询、分页和数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
| `mall-trade` | 佣金和分账的数据隔离需要关联订单、订单商品 |
