# 功能清单-支付退款

## 模块名称

支付退款首批编码模块，对应 `mall-payment`。

## 功能列表

| 功能名称 | 用途描述 | 请求路径 | 入参说明 | 出参说明 |
|----------|----------|----------|----------|----------|
| 查询支付单列表 | 按支付单号、订单号、渠道流水、订单、支付渠道、支付状态筛选支付单 | `GET /api/payment/trades` | `keyword`、`orderId`、`orderNo`、`channelType`、`tradeStatus`、`pageNum`、`pageSize` | 支付单分页列表 |
| 查看支付单详情 | 查看支付单号、订单、支付渠道、支付金额、支付状态、渠道流水和支付时间 | `GET /api/payment/trades/{id}` | 支付单ID | 支付单详情 |
| 查询退款单列表 | 按退款单号、支付单号、渠道退款号、订单、售后单、退款状态筛选退款单 | `GET /api/payment/refunds` | `keyword`、`payNo`、`orderId`、`afterSaleId`、`refundStatus`、`pageNum`、`pageSize` | 退款单分页列表 |
| 查看退款单详情 | 查看退款金额、退款状态、渠道退款号、退款时间和失败原因 | `GET /api/payment/refunds/{id}` | 退款单ID | 退款单详情 |
| 查询资金流水列表 | 按流水号、业务编号、渠道流水、流水类型、方向、主体和状态筛选资金流水 | `GET /api/payment/fund-flows` | `keyword`、`flowType`、`bizNo`、`direction`、`subjectType`、`subjectId`、`status`、`pageNum`、`pageSize` | 资金流水分页列表 |
| 查看资金流水详情 | 查看资金流水事实记录，不提供金额编辑入口 | `GET /api/payment/fund-flows/{id}` | 资金流水ID | 资金流水详情 |

## 业务规则

| 规则 | 说明 |
|------|------|
| 只读起步 | 本批只提供查询能力，不实现发起支付、支付回调、退款回调、退款重试、对账处理、分账和提现打款 |
| 状态边界 | 支付单状态、退款单状态和资金流水状态只追踪资金处理结果，不替代订单、售后、提现和分账业务状态 |
| 资金事实 | 资金流水只记录事实，不允许直接手工改金额，不允许删除历史流水 |
| 失败可见 | 支付失败、退款失败、处理中、关闭等状态必须能在列表中查到，供对账和异常中心追溯 |
| 数据隔离 | 支付单、退款单通过关联订单按城市、自提点、团长、供应商、仓库范围隔离；资金流水按主体范围以及关联支付/退款业务单据隔离 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用 `pay_trade`、`pay_refund_trade`、`pay_fund_flow` 表和初始化数据 |
| `mall-common` | Result、分页、基础实体、全局异常、网关认证头、数据权限拦截器 |
| `mall-api` | 支付单、退款单、资金流水 VO/DTO 契约 |
| MyBatis Plus | 单表查询、分页和数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
