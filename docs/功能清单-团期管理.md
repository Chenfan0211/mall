# 功能清单-团期管理

## 模块名称

团期管理首批编码模块，对应 `mall-sale`。

## 功能列表

| 功能名称 | 用途描述 | 请求路径 | 入参说明 | 出参说明 |
|----------|----------|----------|----------|----------|
| 查询团期列表 | 运营后台团期管理按关键词、大区、城市、采购仓、库存模式、状态筛选团期 | `GET /api/sale/periods` | `keyword`、`regionId`、`cityId`、`warehouseId`、`stockMode`、`status`、`pageNum`、`pageSize` | 团期分页列表 |
| 查看团期详情 | 查看团期销售时间、截团时间、配送日期、库存模式、发布状态和发布人 | `GET /api/sale/periods/{id}` | 团期ID | 团期详情 |
| 查询团期 SKU | 查询团期商品池中的商品/SKU、供应商、售价、采购价、限购、计划库存、已售和锁定数量 | `GET /api/sale/publish-skus` | `periodId`、`productId`、`skuId`、`supplierId`、`status` | 团期 SKU 列表 |
| 查询团期快照 | 查看团期发布、下单、采购、出库等节点的快照证据 | `GET /api/sale/periods/{id}/snapshots` | 团期ID | 团期快照列表 |

## 业务规则

| 规则 | 说明 |
|------|------|
| 团期定位 | 团期是真正可售单元，决定销售时间、价格、库存模式、采购仓、配送日和发布状态 |
| 第一版范围 | 不做团期自提点黑白名单，不配置商品级或团期级售后规则 |
| 状态口径 | 团期状态为 `10待发布 20已发布 30已下架 40已过期` |
| 库存模式 | 库存模式为 `1仓库库存 2虚拟库存`，首批接口只读展示 |
| 数据隔离 | `sale_publish_period` 按城市、仓库范围隔离；供应商通过团期 SKU 关联自身供应商范围隔离；`sale_publish_sku` 按供应商范围隔离 |

## 依赖关系

| 依赖 | 说明 |
|------|------|
| `Sql/mall.sql` | 使用 `sale_publish_period`、`sale_publish_sku`、`sale_period_snapshot` 表和初始化数据 |
| `mall-common` | Result、分页、基础实体、全局异常、网关认证头、数据权限拦截器 |
| `mall-api` | 团期 VO/DTO 契约 |
| MyBatis Plus | 单表查询、分页和数据权限插件 |
| MapStruct | Entity 到 DTO 转换 |
